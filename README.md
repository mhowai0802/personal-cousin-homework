# personal-cousin-homework

A place to organize, track, and work on homework assignments.

## Structure

```
assignments/   # Completed and in-progress homework by subject
notes/         # Study notes and reference material
resources/     # Worksheets, PDFs, and other source files
```

## How to add work

1. Create a folder under `assignments/` named after the subject (e.g. `math`, `english`, `science`).
2. Add each assignment in its own subfolder or file, with the due date in the name when helpful.
3. Put supporting materials in `resources/` and personal notes in `notes/`.
